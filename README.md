# Solar System Simulation (Three.js)

A small 3D project that visualizes the solar system in the browser using **Three.js**.
It shows the Sun at the center with planets orbiting around it, along with a star field background to create a space environment.

## Project Overview

The goal of this project was to understand how 3D scenes work in the browser.
It includes:

* Sun placed at the center of the scene
* Planets revolving around the Sun
* Planet self-rotation
* Orbit paths to show the movement visually
* Background stars to create a space effect
* Interactive camera movement using mouse
